# Chapter 23: Wireless Signals and Modulation

이 장에서는 **무선 신호의 기초** 를 학습합니다. RF(Radio Frequency) 특성, 전파, 변조 방식, 그리고 Wi-Fi 표준을 다룹니다.

무선 네트워크는 보이지 않는 전파를 통해 데이터를 전송합니다. 왜 벽을 지나면 신호가 약해지는지, 왜 5GHz가 2.4GHz보다 빠르지만 범위가 짧은지 이해하면 무선 네트워크 설계와 트러블슈팅이 쉬워집니다.

---

## 1. RF 기초

### 전자기파 (Electromagnetic Wave)

```
무선 신호 = 전자기파

전파 스펙트럼:
┌────────────────────────────────────────────┐
│ 라디오 ← 마이크로파 ← 적외선 ← 가시광 ← X-ray │
│  (Hz)                                      │
│  WiFi는 마이크로파 대역 (2.4GHz, 5GHz)      │
└────────────────────────────────────────────┘
```

### 주파수와 파장

```
주파수 (Frequency): 초당 진동 수 (Hz)
파장 (Wavelength): 파동의 한 주기 길이

관계: 주파수 ↑ = 파장 ↓

2.4GHz: 파장 ≈ 12.5cm (긴 파장, 장애물 통과 용이)
5GHz: 파장 ≈ 6cm (짧은 파장, 장애물 통과 어려움)
```

### 진폭 (Amplitude)

```
진폭 = 신호의 세기

높은 진폭 = 강한 신호 = 더 멀리 전파
낮은 진폭 = 약한 신호 = 짧은 거리
```

## 2. RF 전파 특성

### 자유 공간 경로 손실 (FSPL)

```
FSPL: 거리가 멀어질수록 신호 감쇠

공식: FSPL(dB) = 20log(d) + 20log(f) + 32.44
d: 거리 (km)
f: 주파수 (MHz)

거리 2배 → 6dB 손실
주파수 2배 → 6dB 손실

5GHz가 2.4GHz보다 같은 거리에서 더 많이 손실!
```

### 반사 (Reflection)

```
신호가 표면에서 튕겨 나옴

금속, 유리, 물 → 강한 반사
→ 다중 경로 (Multipath) 발생
```

### 굴절 (Refraction)

```
신호가 매질 경계에서 방향 변경

공기 → 유리: 굴절
→ 신호 경로 변화
```

### 회절 (Diffraction)

```
신호가 장애물 모서리에서 휘어짐

낮은 주파수(2.4GHz): 회절 잘됨 → 장애물 뒤로 전파
높은 주파수(5GHz): 회절 어려움 → 직선 경로 필요
```

### 흡수 (Absorption)

```
신호가 물체에 흡수됨

물, 인체, 나무 → 흡수 높음
금속 → 흡수 낮음 (반사 높음)
```

### 산란 (Scattering)

```
신호가 여러 방향으로 분산

나뭇잎, 빗방울 등 작은 물체
→ 신호 분산, 약화
```

## 3. 신호 측정

### dB (Decibel)

```
dB = 로그 스케일 (비율 표현)

+3dB = 전력 2배
+10dB = 전력 10배
-3dB = 전력 절반

예시:
10mW + 3dB = 20mW
100mW - 10dB = 10mW
```

### dBm (dB-milliwatt)

```
dBm = 1mW 기준 절대 전력

0 dBm = 1 mW
10 dBm = 10 mW
20 dBm = 100 mW
-30 dBm = 0.001 mW

일반적인 AP: 15-20 dBm
```

### dBi (dB-isotropic)

```
dBi = 안테나 이득 (등방성 안테나 대비)

등방성 안테나: 모든 방향 동일 (0 dBi)
지향성 안테나: 특정 방향 집중 (높은 dBi)

예시:
Dipole: 2 dBi
Yagi: 12-15 dBi
Parabolic: 20+ dBi
```

### RSSI (Received Signal Strength Indicator)

```
수신 신호 세기:

> -50 dBm: 매우 강함
-50 ~ -60 dBm: 강함
-60 ~ -70 dBm: 양호
-70 ~ -80 dBm: 약함
< -80 dBm: 매우 약함, 연결 불안정
```

### SNR (Signal-to-Noise Ratio)

```
SNR = 신호 세기 - 노이즈 세기

예시:
신호: -60 dBm, 노이즈: -90 dBm
SNR = 30 dB (좋음)

SNR > 25 dB: 양호
SNR < 15 dB: 불량
```

## 4. Wi-Fi 주파수 대역

### 2.4GHz 대역

```
채널: 1-14 (지역별 상이)
채널 폭: 20MHz
중심 주파수: 2.412GHz - 2.484GHz

비중첩 채널 (20MHz):
[Ch 1]---[Ch 6]---[Ch 11]

장점:
- 넓은 범위
- 장애물 통과 우수
- 레거시 기기 지원

단점:
- 혼잡 (다른 기기: Bluetooth, 전자레인지)
- 3개 비중첩 채널만
```

### 5GHz 대역

```
채널: 36-165 (지역별 상이)
채널 폭: 20/40/80/160MHz
중심 주파수: 5.180GHz - 5.825GHz

UNII 밴드:
UNII-1: 36, 40, 44, 48
UNII-2A: 52, 56, 60, 64
UNII-2C: 100-140
UNII-3: 149-165

장점:
- 많은 비중첩 채널
- 덜 혼잡
- 높은 속도 (넓은 채널)

단점:
- 짧은 범위
- DFS (레이더 회피) 필요
```

### 6GHz 대역 (Wi-Fi 6E)

```
채널: 1-233
채널 폭: 20/40/80/160/320MHz

장점:
- 매우 많은 채널
- 레거시 간섭 없음
- 최대 320MHz 채널

단점:
- 최신 기기만 지원
- 짧은 범위
```

## 5. 변조 (Modulation)

### 변조란?

```
변조 = 데이터를 전파 신호에 실는 방법

디지털 데이터 (0, 1)
       ↓
  변조 (Modulation)
       ↓
아날로그 RF 신호
```

### OFDM (Orthogonal Frequency Division Multiplexing)

```
OFDM: 데이터를 여러 부반송파에 분산

하나의 채널:
[|||||||||||||||||||||||||]
 ↑ 각각이 부반송파

장점:
- 다중 경로에 강함
- 높은 스펙트럼 효율

Wi-Fi 4/5/6 모두 OFDM 사용
```

### OFDMA (Wi-Fi 6)

```
OFDMA: 부반송파를 여러 사용자에게 할당

전통적 OFDM:
시간 1: [User A의 전체 채널]
시간 2: [User B의 전체 채널]

OFDMA:
시간 1: [User A][User B][User C]
        동시에 여러 사용자!

지연 감소, 효율성 향상!
```

### QAM (Quadrature Amplitude Modulation)

```
QAM: 진폭과 위상 변조 조합

QAM 종류 (심볼당 비트 수):
- 64-QAM: 6 bits (Wi-Fi 4)
- 256-QAM: 8 bits (Wi-Fi 5)
- 1024-QAM: 10 bits (Wi-Fi 6)
- 4096-QAM: 12 bits (Wi-Fi 7)

높은 QAM = 높은 속도, but 노이즈에 민감
```

## 6. Wi-Fi 표준

### 802.11 표준 비교

| 표준 | 이름 | 주파수 | 최대 속도 | 채널 폭 |
|------|------|--------|---------|--------|
| 802.11b | - | 2.4GHz | 11 Mbps | 22MHz |
| 802.11a | - | 5GHz | 54 Mbps | 20MHz |
| 802.11g | - | 2.4GHz | 54 Mbps | 20MHz |
| 802.11n | Wi-Fi 4 | 2.4/5GHz | 600 Mbps | 40MHz |
| 802.11ac | Wi-Fi 5 | 5GHz | 6.9 Gbps | 160MHz |
| 802.11ax | Wi-Fi 6/6E | 2.4/5/6GHz | 9.6 Gbps | 160MHz |
| 802.11be | Wi-Fi 7 | 2.4/5/6GHz | 46 Gbps | 320MHz |

### MIMO (Multiple Input Multiple Output)

```
MIMO: 여러 안테나로 동시 전송/수신

SU-MIMO (Single User):
AP ═══ 모든 스트림 ═══ 한 클라이언트

MU-MIMO (Multi User):
AP ═╦═ 스트림 1 ═══ 클라이언트 A
   ╠═ 스트림 2 ═══ 클라이언트 B
   ╚═ 스트림 3 ═══ 클라이언트 C

Wi-Fi 5: DL MU-MIMO (4 사용자)
Wi-Fi 6: UL/DL MU-MIMO (8 사용자)
```

### 빔포밍 (Beamforming)

```
빔포밍: 신호를 특정 방향으로 집중

일반:
     /|\
    / | \
 AP --|--→ 전 방향

빔포밍:
 AP ════════→ 클라이언트 방향 집중

클라이언트 위치 파악 후 신호 집중
→ 더 먼 거리, 더 높은 속도
```

## 7. 정리

### RF 특성 요약

| 특성 | 2.4GHz | 5GHz |
|------|--------|------|
| 범위 | 넓음 | 좁음 |
| 장애물 통과 | 우수 | 약함 |
| 채널 수 | 적음 (3개) | 많음 |
| 간섭 | 많음 | 적음 |

### 신호 측정 요약

| 측정 | 단위 | 의미 |
|------|------|------|
| 전력 | dBm | 절대 신호 세기 |
| 이득 | dBi | 안테나 성능 |
| RSSI | dBm | 수신 신호 세기 |
| SNR | dB | 신호 대 잡음비 |

### 시험 포인트

- dB, dBm, dBi 차이
- FSPL: 거리/주파수 2배 = 6dB 손실
- 2.4GHz vs 5GHz 특성
- OFDM vs OFDMA 차이
- MIMO, MU-MIMO, 빔포밍 개념
- Wi-Fi 세대별 특징
